Welcome to the MDM/R Documentation Hub

This comprehensive documentation suite provides end-to-end technical guidance for all aspects of MDM/R operations. From foundational concepts to advanced troubleshooting, our guides cover the complete lifecycle of meter data management including:

  • Foundational Knowledge: Understanding synchronization fundamentals, file types, and the MDM/R ecosystem
  • Universal SDP ID Management: Requesting and assigning unique identifiers for service delivery points
  • Meter Operations: New installations, meter changes (physical and logical), and configuration updates
  • Parameter Configuration: Setting up critical parameters including VEE Service, CT/PT Multipliers, rate classes, and framing structures
  • Data Synchronization: Transmitting assets, relationships, parameters, and service agreements to MDM/R
  • Billing Processes: Requesting and interpreting billing quantities through XML-based interfaces
  • Troubleshooting: Resolving synchronization errors, IR reports, and correcting crossed meter scenarios

Each guide is designed to help you successfully complete specific scenarios while maintaining compliance with MDM/R Technical Interface Specifications (SME-SPEC-40, SME-SPEC-42). Whether you're performing routine operations or resolving complex data issues, you'll find detailed procedures, checklists, examples, and best practices tailored to your needs.

Select a guide below to get started. New users should begin with the foundational guides to build core understanding before progressing to operational and troubleshooting procedures.
